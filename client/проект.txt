# Языки и технологии, используемые в проекте Keanone Marketplace

## Основные языки программирования
- TypeScript - строго типизированное надмножество JavaScript, используется для клиентской и серверной части проекта
- JavaScript - используется в некоторых вспомогательных скриптах и утилитах

## Фронтенд
- React.js - библиотека для создания пользовательских интерфейсов
- Tailwind CSS - утилитарный CSS-фреймворк для стилизации компонентов
- Wouter - легковесная библиотека для маршрутизации
- Framer Motion - библиотека для создания анимаций
- React Query (TanStack Query) - библиотека для управления состоянием и кэшированием данных
- Zod - библиотека для валидации данных
- React Hook Form - библиотека для работы с формами
- Lucide React - библиотека иконок

## Бэкенд
- Node.js - среда выполнения JavaScript
- Express - веб-фреймворк для создания API
- WebSocket - для реализации чатов в реальном времени
- PostgreSQL - реляционная база данных (предположительно, судя по структуре)

## Сборка и разработка
- Vite - инструмент для сборки и разработки
- npm - менеджер пакетов
- tsx - утилита для запуска TypeScript кода

## Стилизация и анимации
- CSS - для создания стилей
- CSS-переменные - для темизации
- Keyframe-анимации - для создания плавных переходов и эффектов
- CSS-трансформации - для создания интерактивных элементов интерфейса

## Особенности проекта
- Темная тема - принудительная темная тема с тщательно подобранной цветовой схемой
- Адаптивный дизайн - поддержка мобильных и десктопных устройств
- Минималистичные анимации - для улучшения пользовательского опыта
- Многоуровневая система ролей - для управления доступом (OWNER, ADMIN, SHOP_OWNER и т.д.)
- Многоязычность - поддержка нескольких языков интерфейса
